    NAME

    Skryf - Skryf application

    VERSION

    version 1.3

    DESCRIPTION

    easy does it web application development.

    PREREQS

    Perl 5.14+ and Mongo.

    INSTALL

        $ cpanm git@github.com:skryf/Skryf.git
    $ skryf new [NAME]
    $ cd [NAME]
    $ morbo app.pl

    METHODS

    Skryf inherits all methods from Mojolicious and overloads the
following:

    startup

    This is your main hook into the application, it will be called at
application startup. Meant to be overloaded in a subclass.
